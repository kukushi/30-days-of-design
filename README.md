# [2015 年设计长征 主页](http://besike.com/sketch)

# 运行

用 [HarpJS](http://harpjs.com/) 静态服务器。

全局安装 harp：

```
npm install harp -g
```

运行 Web 服务器：

```
harp server --port 9000
```

# 主题

用了 @mdo 的 [Hyde](http://andhyde.com/) 主题。
